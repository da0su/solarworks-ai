# タブ② ： 🔍 CEO確認
**目的：** AIが選んだ案件を「入札する」か「しない」かを決める

---

## タブ②の開き方

```
① 画面上部のタブ行を見る
② 左から2番目の「🔍 CEO確認」と書かれた文字をクリックする
```

クリック後、以下の画面に切り替わります。

```

│  オークション      判定                                            │

│                                                                  │
│  未判断: 199件 ／ 判断済み: 1件   ← 毎日ここを確認                 │
│                                                                  │
│  [🔄 再読込]  ← 左側の黄色いボタン（基本触らない）                 │
│                                                                  │
│  ┌────────────────────────────────────────────────────┐          │
│  │ 🔵 REVIEW  コイン名がここに表示される               │          │
│  │ Heritage Auctions | P3 | 締切: — | Score 75        │          │
│  │                                                    │          │

│  │ ▶ 判断理由を見る                                    │          │

└──────────────────────────────────────────────────────────────────┘
```

---

## 案件を「承認する」手順（入札したい場合）

```
① 画面をスクロールして、上から1番目のカードを見る

② カード内の「▶ 判断理由を見る」という文字をクリックする
   → カードが展開されて根拠テキストが表示される

③ 表示された根拠テキストを読む（数秒）
   例: "NGC MS63 直近3か月中央値¥200,000 / 推定利益率+15.2%"

④ 入札したい場合 → カード左下の「✅ 承認」ボタン（黄色）をクリックする

⑤ 画面に「✅ 承認 + 🎯 入札予定に自動追加しました」と表示されれば完了

⑥ 次のカードへ進む（自動でスクロールされないので、手動でスクロールする）
```

> **承認後は何もしなくてよい。**
> 「入札予定」タブへの追加は自動で行われる。

---

## 案件を「NGにする」手順（入札しない場合）

```
① 上から1番目のカードを見る

② カード内の「▶ 判断理由を見る」という文字をクリックする

③ 理由を読んでNG（入札しない）と判断した場合

④ カード右下の「❌ NG」ボタン（黄色）をクリックする

⑤ 画面が切り替わり、次のカードが表示される（または同じカードが残る）

⑥ 次のカードへ進む
```

> **NGの理由は自動で記録される。** 追加の入力は不要。

## タブ②で「見なくてよい」もの

| 要素 | 無視してよい理由 |
|------|----------------|
| 「オークション」ドロップダウン | キャップが管理する |
| 「判定」ドロップダウン | キャップが管理する |
| 「🔄 再読込」ボタン（左の黄色） | キャップが使う |
| 「未判断: 199件 ／ 判断済み: 1件」の数字 | 参考情報（見るだけ） |
| カード上部の緑色の太いライン | 装飾（機能なし） |
| 「P3」という文字 | 優先度（キャップが設定） |

---

## タブ②でやってはいけないこと

```
❌ 「オークション」や「判定」のドロップダウンを変更しない
   → 表示される案件が変わってしまい、見るべき案件を見逃す

❌ 「🔄 再読込」を押しながら判断しない
   → 判断中にリセットされ、入力が消える

❌ 「▶ 判断理由を見る」を開かずに承認しない
   → 根拠なしで承認することになる

❌ 同じカードの「承認」を2回押さない
   → 入札予定リストに重複して追加される
```

---

## Score（スコア）の読み方

カードに「Score 75」のような数字が表示されます。

```
Score 80以上  → 特に優先して判断する
Score 60〜79  → 通常通りに判断する
Score 60未満  → NGにしても問題が少ない（後回しにしてよい）
```

---

## バッジの意味

カードの左上に色付きのバッジが表示されます。

```
🔵 REVIEW  → AIが「要確認」と判断した案件（CEOの判断が必要）
🟢 OK      → AIが「仕入れ候補」と判断した案件
🟠 CEO判断 → AIが迷っている案件（CEOの最終判断が必要）
```

> どのバッジでも操作は同じ。「承認」か「NG」を押すだけ。

---

## タブ③の開き方

```
│                                                                  │

```

---

## 入札前にオークションページを確認する手順

```
① カードの右端にある「🔗」アイコン（鎖のマーク）をクリックする
   → オークションのページが新しいタブで開く

② 現在価格と締切時間を確認する

③ ブラウザの元のタブに戻る（タブをクリックするか Ctrl+Tab）
```

---

## 入札後に「入札額と結果」を登録する手順

```
① タブ③「🎯 入札予定」を開く

② 該当するコインのカードを探す
   （画面上部から順番に表示されている）

③ カード内の「▶ 💰 入札額を登録 / 結果更新」という文字をクリックする
   → カードが展開されて入力フォームが表示される
```

展開後のフォームはこの形です：

```
┌────────────────────────────────────────────────────────────────┐
│  💰 入札額を登録 / 結果更新（展開中）                             │
│                                                                │
│  入札額(USD)       円換算        入札額(円)                     │
│  [ 827.00 ][ - ][ + ]  ¥133,147  [ 133147 ][ - ][ + ]        │
│    ↑ここに入力      ↑自動計算      ↑触らない                    │
│                                                                │
│  結果:                                                         │
│  ● ⏳予定   ○ 🏆落札   ○ ❌落選   ○ 🚫取消                     │
│  ↑入札前/中  ↑落とせた  ↑負けた   ↑やめた                      │
│                                                                │
│  [             💾 保存             ]  ← 幅いっぱいの黄色ボタン  │
└────────────────────────────────────────────────────────────────┘
```

```
④ 「入札額(USD)」の欄に実際に入札した金額（ドル）を入力する
   例: 830 と入力する

⑤ 「円換算」と「入札額(円)」は自動で計算される（触らない）

⑥ 「結果」のラジオボタンを選ぶ
   ・入札したが結果不明 → 「⏳ 予定」を選ぶ（デフォルト）
   ・落札できた        → 「🏆 落札」を選ぶ
   ・他の人に負けた    → 「❌ 落選」を選ぶ
   ・入札をやめた      → 「🚫 取消」を選ぶ

⑦ 画面下の「💾 保存」ボタン（黄色・横幅いっぱい）をクリックする

⑧ 画面上部に「✅ 保存しました」のメッセージが表示されれば完了
```

---

## 結果が後から判明した場合の更新手順

```
① タブ③「🎯 入札予定」を開く

② 該当カードを探す

③ 「▶ 💰 入札額を登録 / 結果更新」をクリックする

④ 「結果」のラジオボタンを「🏆 落札」または「❌ 落選」に変更する

⑤ 「💾 保存」ボタンをクリックする
```

---

## タブ③の4つの数字の意味

画面上部に表示される4つの数字：

```
🎯 入札予定 ： 現在入札待ちの案件数（承認したが結果未定）
✅ 本日CEO確認 ： 今日 承認orNG した案件数
📅 本日入札 ： 今日 入札登録した件数
🚨 緊急(48h以内) ： 締切まで48時間以内の案件数（赤い数字）
```

> 🚨 が 1以上の場合は、そのカードを優先的に確認する。

---

## タブ③でやってはいけないこと

```
❌ 「入札額(USD)」を空白のまま「保存」しない
   → 金額がゼロとして記録される

❌ 「円換算」の欄を手入力で変更しない
   → 自動計算が正しい値。触ると数字がおかしくなる

❌ 結果が判明する前に「🏆 落札」や「❌ 落選」にしない
   → 不正確な記録になる。判明するまで「⏳ 予定」のままにする

❌ 「🔄 再読込」を入力途中に押さない
   → 入力した内容がリセットされる
```

# タブ① ： 📊 相場DB
**目的：** 過去の落札価格を調べる（必要なときだけ使う）

---

## やること（3ステップ）

```
① 「スラブテキスト / 国名 / 年号で検索」と書かれた黒い入力欄をクリックする
   （画面中央の横長の欄）

② キーワードを入力する
   例:「フランス」「NGC MS63」「1914 Germany」

③ 入力欄右側の「検索」ボタン（黄色）をクリックする
   → 該当コインの一覧が下に表示される
```

---

## 詳細を見たい場合

```
④ コインカードの下部にある「▶ 詳細を見る」という文字をクリックする
   → 直近3か月の落札価格の詳細が表示される
```

---

## タブ①でやってはいけないこと

```
❌ 「+」ボタンを連打してページをたくさん進めない
   → 2,700件以上あり、動作が重くなる

❌ 「🔄 再読込」を頻繁に押さない
   → 検索結果がリセットされる
```

# タブ④ ： 📋 入札実績
**目的：** 全入札の履歴確認（基本はキャップが管理）

---

## やること（3ステップ）

```
① 「📋 入札実績」タブをクリックする

② 画面上部の4つの数字を確認する
   「総入札数 / 🏆落札 / ❌落選 / 総投資額」

③ 実績一覧で各案件の結果を確認する
   （更新はタブ③で行う）
```

---

## 手動で新しく登録したい場合

```
① 「▶ ➕ 新規入力 / スクリーンショット登録」をクリックする
   （画面上部の折りたたみエリア）

② フォームに必要事項を入力する

③ スクリーンショットをドラッグ＆ドロップで添付する（任意）

④ 下部の保存ボタンをクリックする
```

> 通常、タブ③で登録すれば自動でここに反映される。
> 直接入力が必要なのは例外的な場合のみ。

# 📋 毎日の操作を30秒でまとめた版

```
【朝】
Step 1.  http://localhost:8501 をブラウザで開く
Step 2.  「🔍 CEO確認」タブをクリックする
Step 3.  「未判断: ○件」の数字を確認する
Step 4.  1番上のカードを見る
Step 5.  「▶ 判断理由を見る」をクリックして根拠を確認する
Step 6a. 入札したい → 「✅ 承認」ボタン（カード左下・黄色）をクリックする
Step 6b. 入札しない → 「❌ NG」ボタン（カード右下・黄色）をクリックする
Step 7.  スクロールして次のカードに進む
Step 8.  Step 4〜7を繰り返す
Step 9.  「🎯 入札予定」タブをクリックして「🚨緊急」が0件であることを確認する
   └── 1件以上あれば → 右端の「🔗」をクリックして締切時刻を確認する

【入札後すぐ】
Step 1.  「🎯 入札予定」タブをクリックする
Step 2.  該当コインのカードを探す
Step 3.  「▶ 💰 入札額を登録 / 結果更新」をクリックする
Step 4.  「入札額(USD)」欄に実際の入札額（ドル）を入力する
Step 5.  「結果」は「⏳ 予定」のままにする
Step 6.  「💾 保存」ボタン（幅いっぱいの黄色ボタン）をクリックする

【結果判明後】
Step 1.  「🎯 入札予定」タブを開く
Step 2.  該当カードの「▶ 💰 入札額を登録 / 結果更新」をクリックする
Step 3.  「結果」を「🏆 落札」または「❌ 落選」に変更する
Step 4.  「💾 保存」をクリックする
```

# ❓ こんなときどうする

| 状況 | 対処 |
|------|------|
| 画面が真っ白または何も表示されない | F5キーでページを再読み込みする |
| 承認したのに「入札予定」に追加されない | もう一度「CEO確認」タブで同じ案件を「承認」する |
| 「🚨 緊急」が赤くなっている | 入札予定タブで右端「🔗」をクリックして締切を確認する |
| ボタンを押しても何も変わらない | 5秒待ってからもう一度押す |
| 案件が表示されない（0件） | キャップに連絡する |
| 保存したのに消えた | キャップに連絡する |
