# コイン仕入れダッシュボード 操作マニュアル

> **対象:** HR担当者・CEO
> **URL:** http://localhost:8501
> **所要時間:** 初回5分、慣れれば1日1〜2分

---

## 🖥️ 画面全体の構成

ダッシュボードには **4つのタブ** があります。左から順に使います。

```
┌─────────────────────────────────────────────────────────────────┐
│  🏅 コイン仕入れDB                                               │
│  更新: 2026-03-22 | Gold: 23,005円/g | Silver: 347.7円/g | USD: 161 │
│                                                                 │
│  [📊 相場DB]  [🔍 CEO確認]  [🎯 入札予定]  [📋 入札実績]          │
└─────────────────────────────────────────────────────────────────┘
         ①            ②           ③            ④
```

| タブ | 用途 | 毎日使う？ |
|------|------|-----------|
| ① 相場DB | 過去の落札相場を検索・調べる | 必要なときだけ |
| ② CEO確認 | AIが選んだ候補を承認 or NG | **毎日（最重要）** |
| ③ 入札予定 | 入札済み案件の額を登録・結果更新 | 入札後すぐ |
| ④ 入札実績 | 全履歴の確認・新規手動登録 | 確認時 |

## ① 📊 相場DB タブ

### 何をする画面？
過去に落札されたコインの相場価格を検索できます。「このコインは今いくら？」を調べる辞書的な画面です。

### 画面の見方

```
┌─────────────────────────────────────────────────────────────────┐
│  [スラブテキスト / 国名 / 年号で検索              ] [検索]       │
│                                                    ①検索ボックス │
│  ● 全て  ○ Gold  ○ Silver  ○ Platinum                          │
│  ②フィルター                                                    │
│                                                                 │
│  2,701件        [ 1 ][ - ][ + ]      [🔄 再読込]               │
│  ③件数表示    ④ページ移動           ⑤再読込                   │
│  ページ 1/55（50件/ページ）                                       │
│                                                                 │
│  ┌─────────────────────────────────────────────────────┐       │
│  │ NGC                            eBay上限（基準1）      │       │
│  │ 1977 PIEFORT FRANCE G5F        3,371,884円           │       │
│  │ PF 69  Gold                    USD20,943/GBP15,756   │       │
│  │                                                     │       │
│  │ > 詳細を見る                                         │       │
│  └─────────────────────────────────────────────────────┘       │
│  ⑥コインカード（クリックで相場詳細表示）                          │
└─────────────────────────────────────────────────────────────────┘
```

### 操作手順
1. **①** に「フランス」「1977」「NGC PF」などを入力
2. **[検索]** ボタン（黄色）をクリック
3. **⑥ コインカード** の「詳細を見る」をクリックすると直近3か月の相場が表示される

### NG操作
- ❌ **[再読込]を頻繁に押さない** → 画面が再ロードされ遅くなる
- ❌ **ページ移動の `+` を連打しない** → 2,700件以上あり動作が重くなる

## ② 🔍 CEO確認 タブ（最重要）

### 何をする画面？
AIが選んだ仕入れ候補を **承認** または **NG** で判断する画面です。
**承認すると自動で「入札予定リスト」に追加されます。**

### 画面の見方

```
┌─────────────────────────────────────────────────────────────────┐
│  🔍 CEO確認 — 承認/NG判断                                        │
│  status=pending の OK/CEO判断/REVIEW 案件を表示しています。       │
│                                                                 │
│  オークション  [全て ▼]    判定  [全て ▼]                        │
│  ①オークション絞り込み    ②判定タイプ絞り込み                    │
│                                                                 │
│  未判断: 199件 ／ 判断済み: 1件                                   │
│  ③残件数確認（ここを毎日ゼロに近づける）                          │
│                                                                 │
│           [🔄 再読込]                                            │
│                                                                 │
│  ┌─────────────────────────────────────────────────────┐       │
│  │ [REVIEW] George Washington: (1800) GW-70 Funeral... │       │
│  │ Heritage Auctions | P3 | 締切: — | Score 75         │       │
│  │                                                     │       │
│  │  現在価格      仕入コスト     仕入限界                │       │
│  │    —              —              —                  │       │
│  │                                                     │       │
│  │ > 判断理由を見る  ← ④クリックで根拠を展開            │       │
│  │                                                     │       │
│  │ [✅ 承認]          [❌ NG]                           │       │
│  │  ⑤承認ボタン        ⑥NGボタン                       │       │
│  └─────────────────────────────────────────────────────┘       │
└─────────────────────────────────────────────────────────────────┘
```

### バッジの意味

| バッジ | 色 | 意味 |
|--------|-----|------|
| `REVIEW` | 紫 | AIが要確認と判断（CEO判断が必要） |
| `OK` | 緑 | AIが仕入れ候補として承認済み |
| `CEO判断` | 橙 | AIが迷っている案件 |

### Score（スコア）の見方
- **75以上** → 優先度高い（なるべく先に判断）
- **50〜74** → 中程度
- **50未満** → 優先度低い

### 操作手順（1案件あたり）

**STEP 1.** カードの **④「判断理由を見る」** をクリックして根拠を確認

```
展開例:
  利益率未計算
  NGC MS63 直近3か月中央値¥200,000
  推定仕入コスト¥135,000 / 仕入限界¥180,000 / 推定利益率+15.2%
```

**STEP 2.** 内容を見て判断する

- 仕入れたい → **⑤「✅ 承認」** ボタン（黄色）をクリック
- 仕入れない → **⑥「❌ NG」** ボタン（黄色）をクリック

**STEP 3.** 承認した場合 → **自動で「入札予定」タブに追加される**（操作不要）

### NGボタンを押した後

```
┌─────────────────────────────────────────────────────────────────┐
│ > 判断理由を見る（展開中）                                        │
│   利益率未計算                  ← NG理由が表示される              │
│                                                                 │
│ [✅ 承認]          [❌ NG]                                        │
└─────────────────────────────────────────────────────────────────┘
```
NGを押すと自動でNG理由がDB保存されます。追加コメントは不要。

### NG例（こんな案件はNG推奨）

| 状況 | 判断 |
|------|------|
| 仕入限界が — (未計算) | NG（利益計算できていない） |
| Score が 40以下 | NG（優先度が低すぎる） |
| 締切が過去日付 | NG（既に終了） |
| 知らないジャンル（勲章・メダル等） | NG（コイン以外） |

## ③ 🎯 入札予定 タブ

### 何をする画面？
CEO承認済みの案件一覧と、**入札後の結果入力**をする画面です。
承認したら自動追加されているので、「何を入札すべきか」がここで確認できます。

### 画面の見方

```
┌─────────────────────────────────────────────────────────────────┐
│  🎯 入札予定 — 今日の行動リスト                                   │
│  CEO承認 → 自動追加。入札後は入札額を登録するだけ。               │
│                                                                 │
│  🎯 入札予定   ✅ 本日CEO確認   📅 本日入札   🚨 緊急(48h以内)   │
│      1件            0件             1件             0件          │
│  ①予定件数       ②今日確認数     ③今日入札数    ④締切間近       │
│                                                                 │
│  ⚠️ 赤い案件 = 締切24時間以内（最優先で対応）                     │
│                                                                 │
│           [🔄 再読込]                                            │
│                                                                 │
│  ┌─────────────────────────────────────────────────────┐       │
│  │ 1914 A Prussia Germany Gold 20 Mark NGC MS63        │  🔗  │
│  │ eBay | — | 入札: $827                               │ ⑤URL │
│  │                                                     │       │
│  │ > 💰 入札額を登録 / 結果更新  ← ⑥クリックで開く     │       │
│  └─────────────────────────────────────────────────────┘       │
└─────────────────────────────────────────────────────────────────┘
```

### 入札後の結果を登録する手順

**STEP 1.** 案件の **⑥「💰 入札額を登録 / 結果更新」** をクリック

```
┌─────────────────────────────────────────────────────────────────┐
│  💰 入札額を登録 / 結果更新（展開中）                              │
│                                                                 │
│  入札額(USD)    円換算         入札額(円)                         │
│  [ 827.00 ]   ¥133,147      [ 133147  ]                        │
│  A入力欄       B自動計算      C自動計算                           │
│                                                                 │
│  結果:                                                           │
│  ● ⏳予定   ○ 🏆落札   ○ ❌落選   ○ 🚫取消                      │
│  D選択                                                          │
│                                                                 │
│  [         💾 保存         ]   ← Eクリックで保存                 │
└─────────────────────────────────────────────────────────────────┘
```

**STEP 2.** 入力内容を記入する

| 項目 | 入力内容 |
|------|---------|
| A 入札額(USD) | 実際に入札した金額（ドル） |
| B 円換算 | 自動計算される（触らない） |
| C 入札額(円) | 自動計算される（触らない） |
| D 結果 | 入札前は「予定」のまま。落札/落選が判明したら変更 |

**STEP 3.** **E「💾 保存」** ボタン（黄色）をクリック → 完了

### 結果の変更タイミング

| 状況 | 選ぶ結果 |
|------|---------|
| まだ入札前 / 入札中 | ⏳ 予定 |
| 落札できた | 🏆 落札 |
| 他の人に落とされた | ❌ 落選 |
| 入札をキャンセルした | 🚫 取消 |

### 🔗 リンクボタンの使い方
カード右端の **🔗 ボタン** をクリックするとオークションページが開きます。
現在価格・残り時間を確認してから入札してください。

## ④ 📋 入札実績 タブ

### 何をする画面？
全入札案件の履歴確認と、スクリーンショット付きの手動記録をする画面です。

### 画面の見方

```
┌─────────────────────────────────────────────────────────────────┐
│  📋 入札実績 — エビデンス管理                                     │
│  入札した案件の結果を記録・蓄積します。CEOはスクリーンショットを送るだけでOK。 │
│                                                                 │
│  総入札数   🏆 落札   ❌ 落選   総投資額                          │
│    1件        0件      0件        —                              │
│  ①集計                                                          │
│                                                                 │
│  > ➕ 新規入力 / スクリーンショット登録                            │
│  ②手動登録フォーム（通常は使わない）                              │
│                                                                 │
│  ── 実績一覧 ──                                                  │
│           [🔄 再読込]                                            │
│                                                                 │
│  ⏳ 予定  1914 A Prussia Germany Gold 20 Mark NGC MS63    🔗    │
│           eBay | 2026-03-31 | 入札: $827                        │
│  ③実績行（クリックで詳細編集）                                     │
└─────────────────────────────────────────────────────────────────┘
```

### 通常の使い方（CEOは基本これだけ）
1. 入札予定タブで結果を更新する（③ タブ参照）
2. 実績タブはキャップが確認・管理する（CEO操作不要）

### 手動登録が必要な場合（**②** を使う）

**②「➕ 新規入力 / スクリーンショット登録」** をクリックして展開:

| 項目 | 入力内容 |
|------|---------|
| コイン名 | 例: `1914 A Prussia Germany Gold 20 Mark NGC MS63` |
| オークション | 例: `eBay`, `Heritage`, `Spink` |
| URL | オークションページのURL |
| 入札日 | 入札した日付 |
| 入札額(USD) | ドル金額 |
| 結果 | 予定/落札/落選/取消 |
| スクリーンショット | 📎 ドラッグ&ドロップ or クリックでアップロード |

## 🚦 毎日のルーティン（CEOの操作は3分以内）

```
毎朝:
  1. http://localhost:8501 を開く
  2. 「🔍 CEO確認」タブをクリック
  3. 未判断件数を確認 → 上から順に承認 or NG
  4. 終わったら「🎯 入札予定」タブで締切間近の案件を確認

入札後:
  5. 「🎯 入札予定」タブで該当案件を探す
  6. 「入札額を登録/結果更新」を開いて金額・結果を入力
  7. 「保存」をクリック → 完了
```

---

## ⚠️ やってはいけないこと（NG集）

| NG操作 | 理由 |
|--------|------|
| ページを何度もリロードする | データが消える場合がある |
| 同じ案件を複数回「承認」する | 入札予定が重複する |
| 承認前に相場確認しないで押す | 損失リスク |
| 結果が出ていないのに「落選」にする | 履歴が狂う |
| URL欄のある案件で直接相場DBから判断する | CEO確認タブで確認すること |

---

## 🆘 トラブル対応

| 症状 | 対処 |
|------|------|
| 画面が真っ暗 / 表示されない | ページをリロード（F5） |
| 承認ボタンを押したのに消えない | 「再読込」ボタンをクリック |
| 入札予定に追加されていない | CEO確認タブで再度承認する |
| 件数が0件で何も表示されない | キャップに連絡 |
| ボタンを押しても何も起きない | 数秒待ってからもう一度押す |
