## 承認後の操作手順

```bash
# 1. auction_fee_rules.json を開く
# 2. 各社の数値を公式サイト確認値に修正

# 4. confirmed_at に確認日を記入

# 5. 確認コマンド（手数料ルール一覧表示）
cd coin_business
python scripts/auction_cost_calculator.py --list

# 6. 計算テスト（例: Heritage $500, FX=150）
python scripts/auction_cost_calculator.py --calc 500 --source heritage --fx 150 --breakdown

# 7. git commit
```
