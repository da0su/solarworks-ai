# 楽天ROOM 初回テスト投稿手順

**目的**: 投稿プロセス全体を1件で通しテストし、問題点を洗い出す
**使用商品**: アテニア スキンクリア クレンズ オイル（CEO承認済み）

---

## 事前準備チェック

- [ ] 楽天ROOMアカウント（カピバラ癒し）にログインできる
- [ ] 投稿文が準備済み（`posts/2026-03-09_post_final.md` の投稿①）
- [ ] PCブラウザ or スマホアプリが使える状態

---

## テスト手順

### Step 1: ログイン確認（CEO操作）
1. ブラウザで https://room.rakuten.co.jp/ を開く
2. 楽天IDでログイン
3. マイROOMページが表示されることを確認
4. アカウント名「カピバラ癒し」が表示されていることを確認

**確認ポイント:**
- [ ] ログインできた
- [ ] アカウント名が正しい
- [ ] プロフィール画像が表示されている

---

### Step 2: 商品を探す（CEO操作）
1. 楽天市場（https://www.rakuten.co.jp/）を新しいタブで開く
2. 検索バーに「**アテニア スキンクリア クレンズ オイル**」と入力して検索
3. 該当商品のページを開く（レビュー4万件超えの商品）

**確認ポイント:**
- [ ] 商品ページが開けた
- [ ] 正しい商品であることを確認（価格 ¥1,980〜）

---

### Step 3: 「コレ!」で投稿画面を開く（CEO操作）
1. 商品ページ上で「楽天ROOMに投稿」または「コレ!」ボタンを探す
   - ※ページ上部、商品画像の下、または商品情報の近くにある
2. ボタンをクリック
3. 投稿編集画面が表示されることを確認

**確認ポイント:**
- [ ] 「コレ!」ボタンが見つかった
- [ ] 投稿編集画面が開いた
- [ ] 商品画像が自動で設定されている

**もしボタンが見つからない場合:**
- 楽天ROOM側から検索: https://room.rakuten.co.jp/ の検索バーで商品名検索

---

### Step 4: 投稿文を入力（CEO操作）
1. 以下の投稿文をコピーする:

```
夕方の肌のくすみ、地味に気になってた😌

クレンジングって「落とすだけ」って
思ってたけど、これ使ってから
洗い上がりの肌の感じが全然違う...

アテニアのスキンクリア クレンズ オイル、
レビュー4万件超えてるの納得かも。

✔ するっと落ちるのにつっぱらない
✔ 大人のくすみケアもできる
✔ 柑橘系の香りで癒される

1,980円からお試しできるのも嬉しい。
毎日使うものだからこそ
ちゃんと選びたいなって思える一本。

気になる方はROOMに載せてるので
見てみてね🫶

#スキンケア
#クレンジング
#アテニア
#大人美容
```

2. 投稿編集画面のテキストエリアに貼り付ける
3. 貼り付け後、以下を確認:
   - 改行が正しく反映されている
   - 絵文字が表示されている
   - ハッシュタグが正しく入っている

**確認ポイント:**
- [ ] テキストが正しく貼り付けられた
- [ ] 改行が保持されている
- [ ] 絵文字が表示されている
- [ ] ハッシュタグ4個が入っている

---

### Step 5: 投稿前の最終確認（CEO操作）
1. プレビュー機能があれば確認する
2. 以下をチェック:
   - [ ] 商品画像が正しい
   - [ ] レビュー文が読みやすい
   - [ ] ハッシュタグがリンクとして認識されている
   - [ ] 文字数が500文字以内

---

### Step 6: 投稿実行（CEO操作）
1. 「投稿する」ボタンをクリック
2. 投稿完了メッセージを確認

**確認ポイント:**
- [ ] 投稿が完了した
- [ ] エラーメッセージが出ていない

---

### Step 7: 投稿後の確認（CEO操作）
1. 自分のROOMページを開く
2. 投稿が表示されていることを確認
3. 以下をチェック:
   - [ ] マイROOMに投稿が表示されている
   - [ ] レビュー文が正しく表示されている
   - [ ] ハッシュタグがクリック可能になっている
   - [ ] 商品画像・価格が正しい
   - [ ] 商品リンクが楽天市場につながる

---

### Step 8: AIへの結果報告

テスト完了後、CEOは以下をClaude Codeに報告してください:

**報告テンプレート:**
```
楽天ROOM テスト投稿結果:
- 投稿できた: はい / いいえ
- 問題があった: はい(内容を記述) / いいえ
- 「コレ!」ボタンの場所: すぐ見つかった / 探した / 見つからなかった
- 投稿文の貼り付け: 問題なし / 改行が崩れた / 文字化けした
- ハッシュタグ: 正しく反映 / 反映されなかった
- 投稿にかかった時間: 約●分
- その他気づいた点:
```

---

## テスト後の次のアクション

### 問題なしの場合
→ 残り2件（VTシートマスク、マーナ ザルボウル）も投稿
→ 3件投稿が成功したら、Phase 1（30件/日）の準備開始

### 問題ありの場合
→ 問題点をPOSTING_MANUAL.mdのトラブルシューティングに追記
→ 修正後に再テスト

---

## 重要な注意事項

1. **テスト投稿は1件だけ**。まず流れを確認する
2. **急がない**。問題があれば報告してもらえればAIが対処法を提案する
3. **スクリーンショットがあると助かる**。問題があった画面のスクショがあると対応しやすい
4. **投稿時間はいつでもOK**。テスト段階では時間帯ルールは気にしなくてよい
