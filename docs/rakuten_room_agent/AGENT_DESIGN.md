# 楽天ROOM自動投稿エージェント設計書

## エージェント概要
AI会社の最初の実働エージェント。
楽天市場の商品をリサーチし、魅力的な紹介文を生成し、
楽天ROOMに投稿できるフォーマットで出力する。

## 運用モード: 半自動（Phase 1）

```
AI（商品発見 → 投稿作成）
    ↓
CEO確認（OK / 修正指示）
    ↓
投稿
```

## 担当部署

| 部署 | 役割 |
|------|------|
| 09_INTELLIGENCE | 商品トレンド分析・売れ筋データ収集 |
| 02_RESEARCH | 楽天市場の商品リサーチ |
| 05_CONTENT | 紹介文生成・投稿テンプレート作成 |
| 08_AUTOMATION | ワークフロー管理・履歴保存 |
| 00_OPERATIONS | 全体の進行管理 |

## ワークフロー

### Step 1: 商品リサーチ（INTELLIGENCE + RESEARCH）
- 楽天市場のカテゴリ・ランキングから売れ筋商品を発見
- 商品の特徴・価格・レビュー評価を整理
- アフィリエイト報酬率を確認

### Step 2: 商品選定（OPERATIONS → CEO）
- リサーチ結果をCEOに提示
- CEO判断: 投稿する / しない / 別の商品を探す

### Step 3: 紹介文生成（CONTENT）
- 商品の魅力を引き出す紹介文を生成
- ハッシュタグを選定
- 複数パターンの文案を作成

### Step 4: 投稿フォーマット作成（CONTENT + AUTOMATION）
- 楽天ROOM投稿用のフォーマットに整形
- SNS共有用のテキストも同時生成（Instagram / X）

### Step 5: CEO最終確認
- 投稿内容をCEOに提示
- CEO判断: 投稿OK / 修正指示

### Step 6: 投稿・記録（AUTOMATION）
- 投稿履歴を保存
- パフォーマンスデータを記録（後日）

## 成功指標（Phase 1）
- 週5件以上の投稿準備が完了する
- CEOの確認時間が1件あたり1分以内
- 投稿の型が安定する

## Phase 2（将来）
- 完全自動投稿
- パフォーマンスに基づく自動最適化
- SNS同時投稿の自動化
