# 楽天ROOM 1日100投稿 量産体制

## 概要
1日100件の楽天ROOM投稿を効率的に量産する。
CEOの指示1回で、AIが一括でリサーチ→投稿文生成を行う。

## 起動コマンド例
```
楽天ROOM 今日の100件を準備して
```
```
楽天ROOM キッチン30件、美容30件、キッズ20件、ペット20件
```

## ジャンル配分（1日100件の場合）

| ジャンル | 件数 | 曜日で重点変更 |
|----------|------|---------------|
| キッチン・食器 | 15件 | 月曜重点 |
| 美容・コスメ | 20件 | 火曜重点 |
| キッズ・ベビー | 15件 | 水曜重点 |
| スポーツ | 10件 | 木曜重点 |
| ペット | 10件 | 金曜重点 |
| 花・ガーデニング | 10件 | 土曜重点 |
| トレンド・季節もの | 20件 | 常時 |

## 量産フロー

```
Step 1: ジャンル別に楽天ランキングをリサーチ（AI自動）
    ↓
Step 2: 各商品の投稿文を一括生成（AI自動）
    ↓
Step 3: 投稿文一覧をCEOに提示
    ↓
Step 4: CEOが一括OK or 個別修正指示
    ↓
Step 5: 最終版をコピペ用にフォーマット出力
```

## 出力フォーマット

CEOがすぐコピペできるよう、以下の形式で出力：

```
━━━━━━━━━━━━━━━━━━
投稿 #001 / キッチン
━━━━━━━━━━━━━━━━━━
[投稿文をここに]

━━━━━━━━━━━━━━━━━━
投稿 #002 / 美容
━━━━━━━━━━━━━━━━━━
[投稿文をここに]
```

## 効率化のポイント
- 楽天ランキング上位からまとめてリサーチ
- 投稿パターン（A〜E）をランダムに使い分けて飽きさせない
- ハッシュタグはジャンル別にプリセット済み
- 投稿文は300〜400文字で統一
